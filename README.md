## Running Locally


```bash
git clone https://github.com/mahendrakale4/Portfolio-mahendra-kale
cd Portfolio-mahendra-kale
```


Modify `.env` file with your credentials

Then install dependencies and run the development server:
```bash
pnpm install
pnpm dev
```
