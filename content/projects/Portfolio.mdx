# Portfolio Website Project

I created a responsive portfolio website showcasing my skills and projects. The website is optimized for all devices, offering a seamless user experience.

## Technology Stack

- Tailwind CSS
- React.js
- Next.js
- Node.js
- Redis

## Website Structure

The portfolio website comprises the following pages:

1. **Home Page:** Introduces visitors to key information about me.
2. **Experience Page:** Highlights my professional background and achievements.
3. **Projects Page:** Showcases a collection of my notable projects.
4. **Contact Page:** 
   - Connect on [LinkedIn](https://www.linkedin.com/in/mahendra-kale/).
   - Provides a downloadable [resume](https://linktr.ee/mahendrakale).
   - Offers access to my [GitHub](https://github.com/mahendrakale4) profile.

This project allowed me to leverage modern technologies and frameworks to create a visually appealing and functional portfolio.

Feel free to explore the website to learn more about my experiences, projects, and how to get in touch.
